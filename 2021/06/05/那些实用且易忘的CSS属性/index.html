那些实用且易忘的CSS属性

2021年06月05日 781 字 大概 3 分钟

今年跟着B站Pink老师学过CSS基本语法，但平时美化博客时还是会遇到一些CSS相关问题，所以再此记录下。

让a标签铺满整个div

有时候希望点击div同时跳转到对应链接，奈何a的大小取决于文本大小，但可使用以下方法来铺满整个父盒子，前提是div里只有a标签一个内容。

文字颜色渐变

background: -webkit-linear-gradient(...)为文本元素提供渐变背景。

webkit-text-fill-color: transparent使用透明颜色填充文本。

webkit-background-clip: text用文本剪辑背景，用渐变背景作为颜色填充文本。

input 输入框字体样式

placeholder

focus 聚焦

取消 input 的边框

滚动条

  • scroll-track—— 滚动条轨道
  • scrollbar-thumb—— 滚动条滑块
  • scrollbar-button—— 滚动条按钮
  • scrollbar-corner—— 滚动角

文字超出隐藏并显示省略号

white-space 👉 处理元素空白。

常用值有：

  • nowrap 不换行，其他还有 pre-line、pre-wrap 等

word-break 👉 在合适的点断行。

常用值有：

  • break-all 可以在单词内换行
  • keep-all 只能在半角空格或连字符处换行。我们需要超出盒子部分就换行，所以用 break-all。

box-orient 👉 子元素如何排列。

常用值有：

  • vertical 和 horizontal，分别是垂直排列和水平排列。

line-clamp 👉 控制文本行数。

单行（一定要有宽度）

多行
